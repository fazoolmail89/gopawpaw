The http package provides an enhanced client side API to HTTP. Features include cookie based session tracking, access to all HTTP Headers on both the request and response. Currently supports HTTP GET only, POST support may be added ata a later date.

Package Specification

Requires Java 1.2+ from java.sun.com
HyperText Transfer Protocol versions 1.0 and 1.1 can be found at the W3C Http Protocol site.
