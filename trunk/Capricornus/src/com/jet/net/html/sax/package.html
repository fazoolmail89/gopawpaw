This package contains a HTML Parsing API similar to the SAX API for XML. XML Parsers will not parse HTML because the definition of HTML is not nearly as rigid.

Package Specification

Requires Java 1.2+ from java.sun.com
Information on the HyperText Markup Language can be found at the W3C site.
