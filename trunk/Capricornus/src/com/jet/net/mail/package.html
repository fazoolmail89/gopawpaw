The mail package provides an API for client side access to SMTP and POP3 mail servers.

The SMTP API allows for the sending of email messages to existing SMTP servers. It supports multi-part MIMI attachments and basic authentication with AUTH LOGIN.

The POP3 API allows connection to POP3 mail server to retrieve email.

Package Specification

Requires Java 1.2+ from java.sun.com
Information on SMTP can be found in RFC 821 and RFC 822.
Informaiton on POP3 can be found in RFC 1725.
Information on multi-part MIME messages can be found in RFC 2045 and RFC 2046.
