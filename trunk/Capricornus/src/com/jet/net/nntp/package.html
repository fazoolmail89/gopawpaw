The nntp package contains a client side API for communication with Network News Transfer Protocol or NNTP servers. The API does not currently support any form of authentication. Basic data classes are used to encapsulate an Article and Newsgroup. This is a read-only package, no consideration is given to posting articles.

Package Specification

Requires Java 1.2+ from java.sun.com
Information on NNTP can be found in RFC 977.
